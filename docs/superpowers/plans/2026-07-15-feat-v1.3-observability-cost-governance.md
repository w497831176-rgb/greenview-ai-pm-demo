# feat/v1.3-observability-cost-governance Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:executing-plans` for inline execution. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add owner-side session management, per-call traceability, MCP call auditing, token/cost governance, and token-optimization evidence to the YIAI物业 demo.

**Architecture:** Extend the existing SQLite schema with non-destructive migrations for traces, model calls, MCP call audit records, and model pricing. Hook into `app/chat.py` `_stream_agent_response` to emit trace/model/MCP events, persist them, and surface them via new API endpoints. Add a new platform-admin "Observability & Cost Governance" page in `frontend/index.html` with charts, pricing table, budget thresholds, and COST-01/COST-02 comparison flows. Add an end-to-end acceptance script under `scripts/`.

**Tech Stack:** FastAPI, SQLite, Agno, Pydantic, vanilla JS frontend, Tailwind CSS, Chart.js (already loaded in frontend).

## Task 1: Database migrations (non-destructive)

**Files:**
- Modify: `db/property_db.py`

**Changes:**
- Add columns to `chat_sessions`: `title TEXT`, `last_message_at TEXT`, `last_agent TEXT`, `last_message_preview TEXT`.
- Add columns to `chat_messages`: `trace_id TEXT`, `model_id TEXT` (already exists), `status TEXT`, `latency_ms INTEGER`, `error_summary TEXT`.
- Create new table `chat_traces`:
  - `trace_id TEXT PRIMARY KEY`
  - `session_id TEXT NOT NULL`
  - `user_message TEXT`
  - `intent TEXT`
  - `agent_name TEXT`
  - `created_at TEXT`
  - `status TEXT`
- Create new table `model_calls`:
  - `id INTEGER PRIMARY KEY AUTOINCREMENT`
  - `trace_id TEXT NOT NULL`
  - `stage TEXT`
  - `model_id TEXT`
  - `status TEXT`
  - `latency_ms INTEGER`
  - `input_tokens INTEGER`
  - `output_tokens INTEGER`
  - `reasoning_tokens INTEGER`
  - `cached_tokens INTEGER`
  - `total_tokens INTEGER`
  - `usage_source TEXT`
  - `model_selection_reason TEXT`
  - `error_summary TEXT`
  - `price_snapshot TEXT`
  - `estimated_cost_cny REAL`
  - `created_at TEXT`
- Create new table `mcp_call_audits`:
  - `id INTEGER PRIMARY KEY AUTOINCREMENT`
  - `trace_id TEXT NOT NULL`
  - `server_name TEXT`
  - `tool_name TEXT`
  - `arguments TEXT`
  - `status TEXT`
  - `result_summary TEXT`
  - `error_summary TEXT`
  - `latency_ms INTEGER`
  - `created_at TEXT`
- Create new table `model_prices`:
  - `id INTEGER PRIMARY KEY AUTOINCREMENT`
  - `model_id TEXT NOT NULL`
  - `currency TEXT`
  - `effective_date TEXT`
  - `input_price_per_1m REAL`
  - `cached_input_price_per_1m REAL`
  - `output_price_per_1m REAL`
  - `reasoning_price_per_1m REAL`
  - `source_note TEXT`
  - `enabled INTEGER DEFAULT 1`
- Create new table `budget_thresholds`:
  - `id INTEGER PRIMARY KEY AUTOINCREMENT`
  - `per_call_threshold_cny REAL`
  - `daily_threshold_cny REAL`
  - `updated_at TEXT`
